# ☁️ Viettel Digital Talent 2026 - Cloud Track (Season 6)

This repository was created to store and submit personal homework and notes within the framework of the **Viettel Digital Talent 2026 (Season 6) - Cloud Track** program.

## 📌 Purpose
- Keep personal notes from the training sessions.
- Store source code, documents, and submit homework assignments required for each session.

## 📂 Repository Structure
To maintain organization, the content in this repository is structured by learning session. Each directory contains the respective lesson notes and homework for that specific day.
